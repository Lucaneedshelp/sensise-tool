# Sensise Tools Starter

Dieses Starterpaket enthält:

- eine Portal-Startseite
- ein gemeinsames Theme
- eine erste App als JSON-Generator
- eine App-Registry für weitere Kacheln

## Struktur

- `index.html` = Startseite
- `assets/css/` = gemeinsames Styling
- `assets/js/` = Registry und Portal-Logik
- `apps/json-generator/` = erste App

## Veröffentlichung über GitHub Pages

1. Repository anlegen
2. Dateien hochladen
3. Unter **Settings → Pages** als Source den Branch `main` und den Ordner `/ (root)` wählen
4. Nach dem Build die GitHub-Pages-URL aufrufen
